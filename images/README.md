# Extension Icon

## Requirements
- **Filename**: `icon.png`
- **Recommended size**: 128x128 pixels (minimum)
- **Format**: PNG with transparency
- **Location**: Place your icon file in this directory as `icon.png`

## VS Code Icon Guidelines
- The icon will be displayed in the VS Code Extensions Marketplace
- Should be clear and recognizable at small sizes
- Avoid text in the icon (it becomes unreadable when scaled down)
- Use a simple, distinctive design
